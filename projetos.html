Projeto: Dashboard de Loja de Auto Peças

Este projeto consiste no desenvolvimento de um dashboard analítico para uma loja de auto peças, utilizando Ferramenas de BI. O objetivo é fornecer uma visão estratégica das vendas, estoque, clientes e desempenho financeiro da loja, facilitando a tomada de decisões gerenciais.

  • Ferramentas utilizadas: Lovable, Excel, SQL
  • Principais indicadores: Faturamento, Ticket Médio, Produtos mais vendidos, Estoque Crítico, Perfil de Clientes
  • Funcionalidades: Filtros dinâmicos, gráficos interativos, análise temporal, segmentação por categorias de produtos

O dashboard permite identificar oportunidades de negócio, otimizar o controle de estoque e melhorar o relacionamento com os clientes.
